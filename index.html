Boggle

Recreated a digital version of the board game "Boggle." I used the Java Swing library to create the UI and used event listeners to update it upon mouse clicks. Letters that have previously been used for the current word and letters that are not adjacent to the most recently clicked letter are greyed out. At the end of each game, the user's score is compared against the AI to determine who wins.

Technologies:
Java, Swing GUI
